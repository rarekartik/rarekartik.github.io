💡 The Ultimate AI Formatting Prompt

Paste this prompt into your chatbot before asking for notes to ensure perfect formatting:

"Format your response precisely for a PDF parser:
1. Start main questions with Q.
2. Write the ENTIRE conclusion as a single paragraph starting with the word Conclusion:
3. Format main sections (like Characteristics:, Objectives:, Features:) on a new line and end them with a colon (:).
4. For numbered points, write the number and the point (e.g., 1. Focuses on planning). Leave a blank line before its description paragraph.
5. For simple bullet lists, use standard dashes (-)."
